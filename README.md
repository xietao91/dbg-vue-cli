# dbg

## node环境安装
1.安装 [Nodejs](http://nodejs.cn/download/).
2.安装完成后全局安装yarn
```
npm install yarn -g
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
